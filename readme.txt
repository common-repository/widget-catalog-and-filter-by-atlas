Плагин виджета для вывода в сайдбаре терминов произвольной таксономии

== Description ==

1. Плагин работает на архивных страницах произвольной таксономии (выполняется проверка is_tax перед выводом виджета в сайдбаре)
2. Режим отображения разделов и секция зависит от иерархии терминов таксаномии (или один родитель для всех терминов или несколько)


== Installation ==

1. Устанавливается стандартно через Плагины/Добавить новый (в поиске задать Widget Catalog and Filter by Atlas)
2. Активируйте плагин через меню Плагины.
3. Это все.
